<p>Welcome to the Brain games!</p>

<h1>Requirements</h1>
<ul>
<li>The game can only be played on Unix operation system</li>
<li>Node.js</li>
</ul>

<h1>How to install</h1>
<ul>
<li>git clone this repository</li>
<li>cd frontend-project-44</li>
<li>make install</li>
<li>You can play!</li>
</ul>
