<h1>Welcome to the Brain games!</h1>

<h2>Requirements</h2>
<ul>
<li>The game can only be played on Unix operation system</li>
<li>Node.js</li>
</ul>

<h2>How to install</h2>
<ul>
<li>git clone this repository</li>
<li>cd frontend-project-44</li>
<li>make install</li>
<li>You can play!</li>
</ul>
